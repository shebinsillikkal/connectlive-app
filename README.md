# ConnectLive — Real-Time Social Platform

> Group chats, media sharing, reactions and stories — built entirely serverless on Firebase. Scaled to 10K+ concurrent users.

## The Problem
Built a group chat app for a community platform. The real-time stuff works beautifully with Firestore — messages appear instantly, no polling, no delays. Got to 10K users before needing to think about scaling.

## Features
- 💬 Real-time group chats — messages delivered in under 100ms
- 📷 Photo, video and file sharing
- 😀 Reactions, replies and message threads
- 📖 Stories (24-hour disappearing updates)
- 🔔 Push notifications via FCM
- 🔐 Google, Apple and phone OTP sign-in
- 🟢 Online/last seen status

## Stack
```
Flutter | Firebase Firestore | Firebase Auth | Firebase Storage | FCM | Cloud Functions
```
